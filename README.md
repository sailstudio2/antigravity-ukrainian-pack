# Ukrainian Language Pack for Antigravity IDE

Цей пакет забезпечує повну українізацію інтерфейсу користувача для [Antigravity IDE](https://antigravity.google/), включаючи всі стандартні компоненти редактора та унікальні меню й елементи самого середовища Antigravity.

## Особливості
- 🇺🇦 Повноцінний переклад інтерфейсу українською мовою.
- 🚀 Локалізація унікальних складових Antigravity IDE, які відсутні у стандартному VS Code.
- 🔄 Регулярні оновлення разом із новими версіями середовища.

---

## 🚀 Встановлення з Marketplace (Рекомендовано)
1. Відкрийте Antigravity IDE.
2. Перейдіть на вкладку **Extensions** (Розширення) або натисніть `Ctrl+Shift+X`.
3. У полі пошуку введіть `Ukrainian Language Pack for Antigravity`.
4. Натисніть **Install** (Встановити).
5. Після встановлення з'явиться підказка з пропозицією змінити мову та перезапустити редактор. Натисніть **Change Language and Restart** (Змінити мову та перезапустити).

*(Якщо підказка не з'явилась, натисніть `Ctrl+Shift+P`, введіть `Configure Display Language`, виберіть `uk` і перезапустіть IDE).*

## 🛠 Ручне встановлення (з файлу .vsix або папки)

Якщо ви маєте завантажений пакет або архів з розширенням, ви можете встановити його вручну одним із двох способів:

### Спосіб 1. Встановлення з файлу `.vsix`
1. Відкрийте Antigravity IDE.
2. Перейдіть на вкладку **Extensions** (Розширення).
3. Натисніть на іконку меню **"..."** (Views and More Actions) у верхньому правому куті панелі розширень.
4. Виберіть пункт **Install from VSIX...** (Встановити з VSIX...).
5. Знайдіть і виберіть ваш файл `.vsix` на комп'ютері.
6. Перезапустіть редактор.

### Спосіб 2. Встановлення розміщенням у папку розширень
Якщо ви маєте звичайну розпаковану папку з розширенням (наприклад, `antigravity-ukrainian-pack-X.X.X`):
1. Закрийте Antigravity IDE.
2. Скопіюйте цю папку.
3. Перейдіть до системної директорії розширень Antigravity (або VS Code). Зазвичай вона знаходиться тут:
   - **Для Windows:** `C:\Users\Ваше_Ім'я\.antigravity\extensions` (або `~/.vscode/extensions`)
   - **Для Mac/Linux:** `~/.antigravity/extensions` (або `~/.vscode/extensions`)
4. Вставте скопійовану папку в цю директорію.
5. Відкрийте Antigravity IDE.
6. Натисніть `Ctrl+Shift+P` (або `Cmd+Shift+P` на Mac).
7. Знайдіть і виберіть команду **Configure Display Language** (Налаштувати мову відображення).
8. Виберіть **uk** (Українська) у списку.
9. Перезапустіть редактор для застосування змін.
